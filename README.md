# 💧 Controle de Nível de Água

Sistema desenvolvido em Python para simular o monitoramento de um reservatório de água, exibindo mensagens de alerta com cores diferentes conforme o nível da água.

---

## 📌 Objetivo

Demonstrar o uso de listas, funções e bibliotecas externas na criação de um sistema simples de monitoramento, simulando um ambiente real.

---

## 🚀 Funcionalidades

- Monitoramento de 5 níveis de água
- Exibição de mensagens com cores diferentes
- Organização do código com funções
- Simulação automática (sem entrada do usuário)

---

## 🧠 Níveis do Sistema

| Nível | Situação                  | Cor       |
|------|---------------------------|----------|
| 1    | Muito baixo (crítico)     | 🔴 Vermelho |
| 2    | Baixo                     | 🟡 Amarelo |
| 3    | Médio                     | 🟢 Verde |
| 4    | Alto                      | 🔵 Ciano |
| 5    | Muito alto (alerta)       | 🔵 Azul |

---

## 🛠️ Tecnologias utilizadas

- Python
- Biblioteca Colorama
